🎯 推荐方案：使用 Android Studio（最简单）

为什么要用 Android Studio？
→ 自动安装所有需要的工具
→ 界面友好，不容易出错
→ 官方推荐，最可靠

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📋 5 步完成：

【第 1 步】下载
   https://developer.android.com/studio
   点击 "Download Android Studio"
   文件大小约 900MB

【第 2 步】安装
   打开下载的 .dmg 文件
   拖拽 Android Studio 到 Applications
   运行 Android Studio

【第 3 步】首次设置
   选择 "Do not import settings"
   等待 SDK Manager 安装组件（10-15 分钟）
   点击 Next → Finish

【第 4 步】打开项目
   在 Android Studio 中点击 "Open"
   选择：/Users/lijiadong/Documents/file/hide
   等待 Gradle 同步（2-3 分钟）

【第 5 步】构建 APK
   点击菜单：Build → Build APK(s)
   等待 2-3 分钟
   点击 "locate" 查看 APK 文件

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📱 APK 位置：

app/build/outputs/apk/debug/app-debug.apk
