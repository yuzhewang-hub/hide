📋 接下来的步骤：

【步骤 1】创建 GitHub 仓库
   
   1. 访问：https://github.com/new
   2. 仓库名称：hide
   3. 设置为 Private（推荐）
   4. 不要勾选任何选项
   5. 点击 "Create repository"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 2】推送代码

   复制以下命令，将 "你的用户名" 替换为你的 GitHub 用户名：

   git remote add origin https://github.com/你的用户名/hide.git
   git push -u origin main

   然后输入 GitHub 用户名和密码。

   示例：
   git remote add origin https://github.com/lijiadong/hide.git
   git push -u origin main

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【步骤 3】触发构建并下载 APK

   1. 访问你的仓库页面
   2. 点击 "Actions" 标签
   3. 点击 "构建字幕遮挡工具 APK"
   4. 点击 "Run workflow" → "Run workflow"
   5. 等待 3-5 分钟
   6. 在 Artifacts 中下载 APK
